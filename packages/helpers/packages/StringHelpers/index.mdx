## get started 
We provide two way of using this package `single` or `multi` :
```bash
npm i @snappmarket/string-helpers
OR
npm i @snappmarket/helpers
```

## stringReplace ⇒ string
replace a string with another in a text

**Kind**: global variable  

| Param | Type | Description |
| --- | --- | --- |
| find | string | string you want to be replaced |
| replace | string | string you wont to replace |
| string | string | string you want to modify |

<a name="bulkStringReplace"></a>

## bulkStringReplace ⇒ \*
replaces multiple strings in a text

**Kind**: global variable  

| Param | Type | Description |
| --- | --- | --- |
| replacments | object | object of replacement, key is string you want to find, and value is the string you want to replace |
| string | string | string you want to modify |
