## getCookie(key) ⇒ string
Gives us a cookie by name

**Kind**: global function  

| Param | Type | Description |
| --- | --- | --- |
| key | string | name of the cookie you want |

<a name="setCookie"></a>

## setCookie(name, value, expireDays, path)
Sets a cookie

**Kind**: global function  

| Param | Type | Description |
| --- | --- | --- |
| name | string | name of the cookie |
| value | string | value of the cookie |
| expireDays | number | number of the days that cookie could be valid |
| path | string | path of the cookie to set |

| Param | Type | Description |
| --- | --- | --- |
| name | string | name of the cookie you want to delete |

## makeCookieString(cookies) ⇒ string

**Kind**: global function  

| Param | Type | Description |
| --- | --- | --- |
| cookies | object | object of cookies you want to convert too cookie string |
