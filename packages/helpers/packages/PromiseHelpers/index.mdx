## racePromise(timeout) ⇒ Promise.&lt;unknown&gt;
makes a race promise which returns rejection error on timeout

**Kind**: global function  

| Param | Type | Description |
| --- | --- | --- |
| timeout | number | timout in ms to throw an error with resolve |

<a name="makeTimeout"></a>

## makeTimeout(timeout) ⇒ Promise.&lt;unknown&gt;
returns a rejected promise with TIMEOUT message

**Kind**: global function  

| Param | Type | Description |
| --- | --- | --- |
| timeout | number | timout in ms to throw an error with promise reject |
